# Education

Een groeiende verzameling interactieve tools voor het onderwijs.

Live op: https://bartjkamp.github.io/Education/

## Structuur

- `index.html` — Homepage met hero attractor en categoriepanelen
- `klaslokaal.html`, `wiskunde.html`, `schaken.html`, `lesmateriaal.html` — Categorie-landingspaginas
- `about.html` — Over deze site
- `assets/css/shared.css` — Gedeelde styles
- `assets/js/shared.js` — Gedeelde JavaScript (peacock palette, attractor definities, etc.)
- `assets/icons/` — Achtergrond-iconen voor categoriepanelen

## Lokaal openen

Dubbelklik `index.html`. De site werkt zonder webserver.

## Bouwen aan

Pas `assets/css/shared.css` aan om alle paginas tegelijk te restylen.
Pas `assets/js/shared.js` aan om gedeelde JS-logica te wijzigen.
